# OpenGL Train Game

## 1. Description

* Fully 3D Train Game implemented in OpenGL
* The player must deliver requests to a central station by building rails between stations and deploying trains
* Features:
    * Robust rail editor allowing for straight and curved rails between any two points
    * Branching rails
    * Changeable track switches
    * Map with 3 types of land (grass, water and mountain) which change how rails look (bridges and tunnels)
    * Detailed minimap showing the position, orientation and resource carried by all trains as well as all stations and rails
    * Train editor allowing for multiple cars and resource selection
    * Resource processing at stations

## 2. How to Install
* Download gfx framework (https://github.com/UPB-Graphics/gfx-framework)
* Replace everything in src with the contents of the repo
* Run the game using Visual Studio/Visual Studio Code alongside Cmake
